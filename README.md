# ROS 2 + Dobot MG400

Проєкт складається з двох основних частин: роботи з реальним роботом Dobot MG400 та його моделі. 
Тут надано код для симуляції з ROS 2 в Gazebo. 

Попередньо я вивчала можливості реальної роборуки, щоб краще зрозуміти, як вона повинна поводитись у симуляції. Використовуючи графічну мову, вбудовану в IDE, я навчила робота малювати логотип кафедри.​

- 'mg400_controller': Відповідає за налаштування контролерів для керування рухом робота.
- 'mg400_thesis': Містить файли для запуску симуляції в Gazebo.
- 'mg400_description': Містить опис робота у форматі URDF/XACRO. Ці файли для візуалізації взяті з https://github.com/HarvestX/MG400_ROS2.git

# ROS 2 + MG400

Робота сфокусована на освоєнні кобота Dobot MG400 як у симуляції з ROS 2, так і в реальності.

Для візуалізації роборуки в симуляції використовувала деякі файли з https://github.com/HarvestX/MG400_ROS2.git (додала їх сюди)

- 'mg400_controller': Відповідає за налаштування контролерів для керування рухом робота.
- 'mg400_thesis': Містить файли для запуску симуляції в Gazebo.
- 'mg400_description': Містить опис робота у форматі URDF/XACRO.
